## Troubleshooting

### Spa Not Heating?
1. Check that it's turned on at control panel
2. Verify temperature setting
3. Allow time - can take hours to heat
4. Contact property manager if issue persists

### Jets Not Working?
1. Check if jets are in "off" cycle
2. Press jets button to cycle through settings
3. Check that air valves are open
4. Contact property manager if still not working

### Error Code on Display?
1. Note the error code
2. Try resetting: turn off for 30 seconds, turn back on
3. Contact property manager with error code

### Water Appears Cloudy?
- Contact property manager immediately
- Do not use until cleared for use
- May need chemical adjustment

### Cover Won't Close Properly?
- Check for obstructions
- Ensure hinges aren't blocked
- Don't force it - contact property manager
