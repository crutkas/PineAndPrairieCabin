## Smart Home Tips

### Do's
- Feel free to adjust temperature for comfort
- Use voice commands for convenience
- Connect your devices to WiFi
- Ask the voice assistant for help

### Don'ts
- Don't reset any smart devices to factory settings
- Don't change WiFi network settings
- Don't share WiFi password publicly
- Don't disconnect or move smart home devices

## Troubleshooting

### WiFi Not Working?
1. Check if other devices can connect
2. Restart your device
3. Restart router (unplug for 30 seconds, plug back in)
4. Router location: [Location]
5. Contact property manager if issues persist

### Smart Device Not Responding?
1. Try voice command again
2. Check if device is plugged in
3. Use physical controls as backup
4. Contact property manager for assistance
