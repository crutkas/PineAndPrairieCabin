## Image Specifications

### Recommended Sizes
- **Width:** 1200-2000 pixels
- **Height:** 800-1200 pixels (landscape orientation preferred)
- **Format:** JPEG (.jpg) for best compatibility
- **Quality:** High resolution but optimized for web (under 500KB per image)

### Photography Tips
- Use natural lighting when possible
- Shoot during golden hour for exterior shots
- Show the cabin's best features and views
- Include mountain/forest backdrop to emphasize location
- Keep spaces clean and uncluttered
- Consider professional photography for main images
