## Common Issues & Solutions

### "No Signal" on Screen
- Press "Input" or "Source" button on TV remote
- Select the correct HDMI input (usually HDMI 1 or 2)
- Ensure streaming device is powered on

### Remote Not Working
- Check batteries
- Point directly at TV/device
- Remove any obstructions
- Backup remotes in [location]

### App Not Loading
- Close and reopen the app
- Check WiFi connection
- Restart streaming device (unplug for 30 seconds)
- Restart TV

### Can't Find a Channel
- Try voice search: "[Show name]"
- Check multiple streaming apps
- Use TV guide (if cable available)

### Sound Issues
- Check TV volume isn't muted
- Check soundbar is powered on
- Verify correct input on sound system
- Check app isn't muted

## Screen Mirroring / Casting

### From iPhone/iPad (AirPlay)
1. Ensure device is on same WiFi network
2. Swipe down to Control Center
3. Tap "Screen Mirroring"
4. Select "[TV Name]"
5. Enter code if prompted

### From Android (Chromecast)
1. Ensure device is on same WiFi network
2. Open app you want to cast from
3. Tap cast icon
4. Select TV from list
